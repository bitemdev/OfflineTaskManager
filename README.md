# Offline Task Manager

A lightweight, local-only task manager board designed for solo developers and creators. No cloud, no subscriptions, no bloat. Just your projects and tasks saved locally to your machine.

## Features
* **100% Offline:** All data is saved locally via JSON.
* **Auto-saving:** Changes are written to disk instantly.
* **Project Containers:** Organize your workflow into separate projects.
* **Classic 3-Column Layout:** TO DO, IN PROGRESS, DONE.
